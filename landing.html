Music Player

A simple browser-based music player. Upload your local songs and control playback — no installs needed.

Open Player →

Features

  • Upload and play local audio files
  • Play, Pause, Next, Previous controls
  • Live progress bar
  • Loop a single song or a set of songs
  • Displays the current song name

About

This project was built as an assignment using plain HTML, CSS, and JavaScript — no frameworks. I learned how the HTML5 Audio API works, how to handle local files with the File API, and how to keep the UI in sync with audio events.
