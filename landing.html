Music Player

A simple browser-based music player. Upload your local songs and control playback — no installs needed.

Open Player →

Features

  • Upload and play local audio files
  • Play, Pause, Next, Previous controls
  • Live progress bar
  • Loop a single song or a set of songs
  • Displays the current song name

About

This project was built as an assignment using plain HTML, CSS, and JavaScript. I learned how browsers handle local files through the File API. When a user selects files, you don't get a file path — you get File objects. I used URL.createObjectURL() to turn those into temporary URLs the audio element can play, without needing to upload them anywhere. This keeps everything client-side and private.
